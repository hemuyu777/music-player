一个基于 **Python + PyQt** 开发的桌面音乐播放器。  
项目主要用于学习 PyQt 图形界面开发、多媒体播放功能实现，以及桌面应用的基本结构设计。

---

## 📌 项目简介

本项目是一个本地桌面音乐播放器，提供了基础的音乐播放功能和图形化界面。  
用户可以通过界面完成歌曲播放、暂停、切换、播放模式调整等操作。

这个项目适合作为：

- PyQt 课程设计 / 作业项目
- Python GUI 学习练手项目
- 桌面音乐播放器入门项目
- GitHub 项目展示作品

Python 3.x
PyQt5
Windows 10 / 11
Git
程序启动后，通常可以完成以下操作：

打开音乐播放器界面
加载本地音乐资源
点击播放按钮开始播放
使用暂停、上一首、下一首等按钮控制播放
根据界面设置切换不同播放模式
